1. What components are part of the Von Neuman Architecture

The Von Neuman Architecture has three distinct components: Central Processing Unit (CPU), Memory and Input/output (I/O)

2. What is user space?

User space is system memory allocated to running applications.

3. What is the shell?

A Shell provides you with an interface to the Unix system. Shell is a powerful tool for manipulating data and executing several applications under certain conditions

4. Where is the shell located?

The shell is located in the "user space", and can be used by applications like bash or sh.

5. What is the main task of the shell?

Shell's primary purpose is to read commands and run other programs.

6. What is the octal representation of a file that can be read and written only by its owner?

octal = 6 0 0 (symbolic = rw- --- --- | 3-bit = 110 000 000)

7. In UNIX systems, what is the common name of the special user that can access and modify every file?

"SUDO" or "Super User Do" is the common name of the special user

8. Which file permission classes are available in UNIX systems?

u (user), g (group) and o (other). You can also write a (all)

9. As of 2020, which percentage of servers have UNIX-like OS?

As of 2020 unix like os is used by more than 70% of the servers
